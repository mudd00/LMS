# 🎮 MetaPlaza 기능 사용 가이드

## 👨‍💼 관리자 페이지

### 관리자 계정 정보
- **이메일**: `admin@admin.com`
- **비밀번호**: `admin123`

### 접속 방법
1. 메인 화면에서 "Click to Start" 클릭
2. "로그인" 버튼 클릭
3. 관리자 계정으로 로그인
4. **자동으로 관리자 페이지(`/admin`)로 이동**
   - 일반 사용자: 게임 화면으로 진입
   - 관리자: 관리자 페이지로 자동 리다이렉트

### 관리자 페이지 기능

#### 📊 대시보드 (메인)
- 전체 사용자 수
- 오늘 가입자 수
- 활성 사용자 수
- 신고 건수
- 최근 활동 내역
- 시스템 알림

#### 🔧 관리 메뉴 (좌측 사이드바)
1. **사용자 관리** - 회원 정보 조회/수정/삭제
2. **신고 관리** - 신고 내역 확인 및 처리
3. **공지사항 관리** - 공지 작성/수정/삭제
4. **채팅 로그** - 채팅 내역 모니터링
5. **게시판 관리** - 게시글/댓글 관리
6. **게임 방 관리** - 활성 방 모니터링
7. **상점 관리** - 상품 등록/수정
8. **결제/환불** - 결제 내역 및 환불 처리
9. **통계** - 접속/매출 통계
10. **감사 로그** - 관리자 활동 기록
11. **시스템** - 서버 상태/설정

#### 🚪 로그아웃
- 우측 상단 "로그아웃" 버튼 → 초기 화면으로 복귀

## 🎯 일반 사용자 기능

### 회원가입 및 로그인
1. **회원가입**:
   - "Click to Start" → "회원가입"
   - 닉네임, 이메일, 비밀번호 입력
   - 회원가입 완료 → 초기 화면 복귀

2. **로그인**:
   - "Click to Start" → "로그인"
   - 이메일, 비밀번호 입력
   - 로그인 성공 → 3D 게임 환경 진입

### 캐릭터 조작
- **이동**: `W` `A` `S` `D` 또는 방향키
- **달리기**: `Shift` + 이동키
- **디버그**: `C` 키 (카메라 위치 로그)

### 게시판 사용
1. 우측 상단 **게시판 아이콘 (📋)** 클릭
2. **사이드바** 메뉴:
   - 게시판 / 공지사항 전환
   - 햄버거 메뉴로 숨김/표시
3. **글쓰기**:
   - "글쓰기" 버튼 클릭
   - 제목, 내용, 이미지 URL 입력
4. **게시글 보기**:
   - 게시글 클릭 → 상세보기
   - 추천/비추천
   - 댓글 작성

### ESC 메뉴
- **ESC 키** 누르기
- **계속하기**: 게임 복귀
- **설정**: 준비 중
- **로그아웃**: 초기 화면 복귀

### 지도 기능
- **지도 열기**: 우측 상단 "Open Map" → Mapbox 3D 지도
- **지도 닫기**: 좌측 상단 "Back" → 기본 환경 복귀

## ⚠️ 주의사항

### 모달 사용 시
- 게시판, ESC 메뉴 등 모달이 열려있으면 **캐릭터 이동 자동 비활성화**
- 모달을 닫으면 이동 기능 다시 활성화

### 권한 구분
- **일반 사용자 (ROLE_USER)**: 게임 환경만 이용 가능
- **관리자 (ROLE_DEVELOPER)**: 관리자 페이지 전용 접근
